# SF Waiter AI Bot

MVP Telegram-бот для обучения официантов меню и продажам.

## Что умеет
- регистрация сотрудника
- изучение меню по категориям
- карточки блюд с логикой продажи
- тесты по меню
- экзамен дня
- AI-тренировка продаж
- отчёт по сотруднику
- админ-сводка по ресторанам

## Меню
Структура категорий адаптирована по публичному сайту SF Shaurma Food:
- Акции / Комбо
- Шаурма
- Бургеры
- Пицца
- Роллы
- Шашлыки
- Завтраки
- Супы
- Восточная кухня
- Салаты
- Стейки и горячее
- Детское меню
- Поке & Удон
- Закуски и гарниры
- Соусы
- Десерты & Блины
- Напитки

## Как запустить

### 1. Создай бота
Через BotFather в Telegram:
- `/newbot`
- задай имя
- получи `TELEGRAM_BOT_TOKEN`

### 2. Установи Python зависимости
```bash
pip install -r requirements.txt
```

### 3. Создай `.env`
Скопируй `.env.example` в `.env` и вставь ключи.

### 4. Запусти
```bash
python main.py
```

## Как включить AI-оценку
Если `OPENAI_API_KEY` не задан, бот работает в режиме локальной оценки по ключевым признакам.
Если ключ есть — использует OpenAI для разбора ответов официанта.
