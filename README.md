# SF Training Bot — static training build

Стабильная версия без автосинхронизации с сайтом.

## Для кого
- официанты
- кассиры
- администраторы
- директора

## Что внутри
- полная локальная база меню для обучения
- отдельный блок по акциям / комбо / сетам
- академия продаж по категориям
- карточки блюд
- глубокий разбор позиции
- тесты по меню
- экзамен дня
- тренировка дополнительных продаж
- прогресс сотрудника
- админ-сводка

## Railway
1. Замени файлы проекта
2. Проверь Variables:
   - TELEGRAM_BOT_TOKEN
   - OPENAI_API_KEY (необязательно)
3. Redeploy
4. Запусти бота

## Важно
Эта версия не зависит от сайта в реальном времени.
Она нужна, чтобы бот работал стабильно уже сейчас.
Дальше меню можно обновлять вручную через `sample_menu.json`.
