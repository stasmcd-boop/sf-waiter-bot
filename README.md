# SF Academy Bot v2

Более user-friendly версия бота для обучения официантов.

## Что нового
- улучшенная навигация:
  - 🍽 Меню
  - 🎁 Акции
  - 🎓 Академия меню
  - 🧪 Тесты
  - 💬 Продажи AI
  - 📈 Прогресс
  - 🛠 Админ
- отдельный учебный режим по категориям
- глубокий разбор блюд
- мини-вопросы по конкретным позициям
- команда `/sync_site` для синхронизации меню с сайтом
- команда `/menu_stats` для проверки наполнения меню

## Как обновить меню с сайта
1. Запусти бота
2. Зайди под ролью "директор" или "администратор"
3. Отправь команду:
   `/sync_site`

Бот попробует собрать категории и позиции прямо с сайта `SF_SITE_URL`.

## Важно
Синхронизация с сайтом реализована как практический scraper по публичным страницам сайта.
Если на сайте изменится разметка, синхронизацию может понадобиться чуть подправить.
Но это уже гораздо лучше, чем держать всё вручную в JSON.

## Railway
После замены файлов просто сделай redeploy.
