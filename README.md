# SF Academy Bot v5

Версия без sitemap.

## Как синхронизируется меню
Бот проходит по заранее заданным категориям сайта SF Shaurma Food и:
1. открывает страницу категории `/menu/<category>`
2. вытаскивает товарные ссылки `/menu/<category>/<item>`
3. открывает карточки товаров и собирает название, вес, цену и описание

## Что сделать в Railway
1. Замени файлы
2. Убедись, что в Variables есть:
   - `TELEGRAM_BOT_TOKEN`
   - `SF_SITE_URL=https://shaurma-food.kz`
   - `OPENAI_API_KEY` (опционально)
3. Redeploy
4. В боте зайди под ролью директор/администратор
5. Выполни `/sync_site`
6. Проверь `/menu_stats`

## Команды
- `/sync_site` — обновить меню с сайта
- `/menu_stats` — показать, сколько категорий и позиций подтянулось
